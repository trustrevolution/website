# Trust Revolution

Podcast website for [Trust Revolution](https://trustrevolution.co).

## Stack

- [Hugo](https://gohugo.io) static site generator
- [Netlify](https://netlify.com) hosting
- [Fountain](https://fountain.fm) for podcast distribution

## Development

```bash
hugo server -D
```

## Build

```bash
hugo --gc --minify
```
